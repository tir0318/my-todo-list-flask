# 📝 Modern Todo List App

シンプルでモダンなデザインのタスク管理アプリケーションです。PythonのWebフレームワークであるFlaskを使用して開発しました。
データベース設定が不要で、テキストファイルベースで手軽に動作するのが特徴です。

## ✨ 主な機能

* **タスク管理 (CRUD)**: タスクの追加、表示、編集、削除が可能です。
* **完了状態の切り替え**: ワンクリックでタスクの完了/未完了を切り替えられます。
* **データ永続化**: タスクデータはテキストファイル（`tasks.txt`）に自動保存されるため、アプリを閉じてもデータは消えません。
* **モダンなUI/UX**:
* OSの設定に合わせた**ダークモード**の自動適用。
* 削除時の確認ダイアログによる誤操作防止。
* 直感的な操作が可能なレスポンシブデザイン。



## 🛠 使用技術

* **Backend**: Python 3, Flask
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
* **Template Engine**: Jinja2
* **Storage**: Text file (CSV-like format)

## 🚀 インストールと実行方法

### 1. リポジトリのクローン

```bash
git clone https://github.com/あなたのユーザー名/リポジトリ名.git
cd リポジトリ名

```

### 2. 依存ライブラリのインストール

Pythonがインストールされている環境で、Flaskをインストールします。

```bash
pip install flask

```

### 3. アプリケーションの起動

```bash
python todo_app/app.py

```

### 4. ブラウザでアクセス

ブラウザを開き、以下のURLにアクセスしてください。
`http://127.0.0.1:5000/`

## 📂 ディレクトリ構成

```text
.
└── todo_app/
    ├── app.py              # アプリケーションのエントリーポイント & ロジック
    ├── tasks.txt           # データ保存用ファイル（自動生成されます）
    ├── static/             # 静的ファイル
    │   ├── style.css       # デザイン・ダークモード設定
    │   └── script.js       # インタラクション制御
    └── templates/          # HTMLテンプレート
        ├── base.html       # 共通レイアウト
        ├── index.html      # タスク一覧・追加画面
        └── edit.html       # タスク編集画面

```

## 💡 工夫した点

* **データベースレス**: 大掛かりなデータベースサーバーを立てることなく、Python環境さえあればすぐに動かせるようにファイル入出力でデータを管理しています。
* **ユーザビリティ**: ページを開いた瞬間にタスク入力欄にフォーカスが当たるようにし、キーボードだけでスムーズにタスク追加ができるように調整しました。
* **デザイン**: CSS変数（Variables）を活用し、メンテナンス性の高いスタイルシート設計を行っています。
